Projects

Data Visualization Project: U.S. Tornado Visualization Dashboard

Developed an interactive data visualization tool using Streamlit and Altair to analyze tornado trends across U.S. states over time. Integrated NOAA data and geospatial filtering capabilities.

President Speech Religious Reference Analyzer

Built a sentence-level BERT classification pipeline to distinguish references to religious doctrine vs. institutions in presidential speeches. Incorporated SHAP-based interpretability and temporal trend visualizations.

Custom NLP Pipeline for Religious Content Classification

Created a hybrid keyword + transformer-based model to classify speech content into doctrine, institution, or neutral, with manual annotations and rule-based preprocessing for initial training.
