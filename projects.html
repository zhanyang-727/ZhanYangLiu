Projects

MSI Capstone Project: Electric Vehicle Sales Visualization

This project visualizes trends in electric vehicle (EV) sales across the United States using Streamlit. I designed a geographic and temporal interface for exploring patterns by state and year.

Streamlit Data Visualization Geographic Analysis

Tornado Event Visualization Dashboard

As part of a collaborative project, I built multiple modules in a Streamlit dashboard to visualize tornado patterns in the U.S. — including monthly frequency, size distribution, and state-wise intensity. The dashboard offers interactive plots built with Altair.

Streamlit Altair Collaborative Project

Religious Framing Classifier in Political Discourse

This project applies NLP techniques to classify religious references in political speech as either emphasizing group identity or personal religiosity. Using a fine-tuned sentence-BERT model and SHAP for interpretability, I built a full pipeline from data annotation and preprocessing to model training and analysis.

NLP BERT SHAP Political Science

Political Opinion Clustering from Election Data in the Middle East

Applied unsupervised learning (PCA, K-means) to election survey data to uncover clusters in political attitudes among voters in the Middle East. Generated clean visual summaries of each latent group.
