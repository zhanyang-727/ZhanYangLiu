Tornado Event Visualization Dashboard

As part of a collaborative project, I built multiple modules in a Streamlit dashboard to visualize tornado patterns in the U.S. — including monthly frequency, size distribution, and state-wise intensity. The dashboard offers interactive plots built with Altair for comprehensive meteorological data exploration.

Religious Framing Classifier in Political Discourse

This project applies natural language processing techniques to classify religious references in political speech as either emphasizing group identity or personal religiosity. Using a fine-tuned sentence-BERT model and SHAP for interpretability, I built a comprehensive pipeline from data annotation and preprocessing to model training and analysis.

Political Opinion Clustering from Election Data in the Middle East

Applied unsupervised learning techniques (PCA, K-means) to election survey data to uncover distinct clusters in political attitudes among voters in the Middle East. Generated clean visual summaries of each latent group, revealing underlying patterns in political preferences and demographic correlations.
